# Smart Parking - Frontend UI (Static Demo)

Open `index.html` in a modern browser. No build step needed.

## Features
- Light/Dark mode toggle (persisted)
- Search and filters (lot/type/status)
- Slots grid with Book/Release
- Booking dialog with live total; demo revenue counter
- Mini map visualization and export demo data

## Run locally (optional)
```bash
# PowerShell
python -m http.server 8000
# visit http://localhost:8000
```

## Files
- `index.html` – layout
- `assets/styles.css` – styles
- `assets/app.js` – interactions + demo data
